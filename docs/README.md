## 🎯 快速开始

### 对于新开发者
1. 先阅读 [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md) 了解项目结构
2. 再查看 [PROGRESS.md](PROGRESS.md) 了解当前进度
3. 然后看 [TODO.md](TODO.md) 找到可以开始的任务
4. 需要时参考 [design.md](design.md) 和 [api-design.md](api-design.md)

### 对于项目管理者
1. 每周查看 [PROGRESS.md](PROGRESS.md) 了解进度
2. 每日查看 [TODO.md](TODO.md) 跟踪任务
3. 参考 [plan.md](../plan.md) 调整计划

### 对于架构师
1. 重点阅读 [design.md](design.md) 系统设计
2. 参考 [api-design.md](api-design.md) API 设计
3. 查看 [tech-stack.md](../tech-stack.md) 技术选型
